# data-structures-impl

A collaborative implementation of core tree-based data structures in Python and Java, designed to deepen understanding of data organization, traversal, and efficient searching.

## Implementations

- Tree Traversals
  - In-order
  - Pre-order
  - Post-order
  - Level-order

- Binary Search Tree (BST)
  - Insertion
  - Deletion
  - Search

- AVL Tree
  - Self-balancing operations
  - Rotations (LL, RR, LR, RL)

- B-Tree
  - Multi-way search tree
  - Node splitting and balancing

- B+ Tree
  - Optimized for range queries
  - Leaf node linkage

## Tech Stack

- Python
- Java

## Objectives

- Understand internal mechanics of tree-based data structures
- Compare performance and behavior across implementations
- Practice clean, modular coding in a team setting
